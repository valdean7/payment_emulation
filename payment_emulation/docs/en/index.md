# Home

## Description
`payment_emulation` is a library developed to be used in Django projects to 
emulate bank card payments. It is ideal for use in your projects to implement a 
payment method.

## Requirements
Payment emulation requires the following:

- Django (>=5.1)
- pycpfcnpj (>=1.8)
- Python (>=3.10)
